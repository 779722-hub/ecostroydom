## Как сохранить

1. Открой каждое сообщение в чате и сохрани картинку (правой кнопкой → «Сохранить изображение как…»).
2. Положи в `C:\Users\user\ecostroydom\assets\img\` под именем из колонки **Имя файла**.
3. Перезагрузи `index.html` в браузере — все картинки появятся.
