# Управление акцией / таймером

Файл: `config.js`

Раз в **3 дня** менеджер обновляет дату окончания акции:

1. Открой `assets/js/config.js`
2. Замени значение `endsAt` на новую дату (текущая дата + 3 дня), формат:
   `"2026-04-16T23:59:59+05:00"`
3. Сохрани и залей файл по FTP / в репозиторий.

При истечении таймера автоматически появится сообщение «Акция продлена — уточните условия у менеджера», а сам отсчёт обнулится. После обновления `endsAt` таймер снова заработает.

Дополнительно можно менять:
- `discount` — текст скидки
- `bonus` — текст подарка
- `durationDays` — длительность акции (для расчёта прогресс-бара)
