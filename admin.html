Админка акции

Быстрое редактирование баннера акции на ecostroydom.kz. Изменения появятся на сайте через ~1 минуту (сборка GitHub Pages).

Токен хранится только в вашем браузере (localStorage). Создать: github.com/settings/tokens → выбрать scope repo.

Параметры акции

Служебное
Пишем в ветку main, файл assets/js/config.js. После коммита GitHub Pages пересобирает сайт за 30-90 секунд. Обновите вкладку сайта Ctrl+F5 чтобы увидеть новый текст. Таймер читает endsAt раз в секунду и обновится сразу при следующем рендере.
